# ChefDK 2.2.1 Release Notes

The 2.2.1 release includes RubyGems 2.6.13 to fix the following CVEs:
  * CVE-2017-0899
  * CVE-2017-0900
  * CVE-2017-0901
  * CVE-2017-0902

ChefDK 2.2 includes:
  * Chef 3.3.42
  * InSpec 1.35.1
  * Berkshelf 6.3.1
  * Chef Vault 3.3.0
  * Foodcritic 11.3.1
  * Test Kitchen 1.17.0
